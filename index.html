Open source projects


ProxSDP

ProxSDP is an open-source semidefinite programming (SDP) solver developed entirely in the Julia language. The main advantage of ProxSDP over other state-of-the-art solvers is the ability to exploit the low-rank structure inherent to several SDP problems.

StateSpaceModels.jl

StateSpaceModels.jl is a package for modeling, forecasting, and simulating time series in a state-space framework.
